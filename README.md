# InfaDebloater V1

## Usage:
1. Mod app_list.txt
2. Chmod
```
su -c chmod +x debloater.sh
```
### Enable
```
su -c sh debloater.sh enable app_list.txt
```
### Disable 
```
su -c sh debloater.sh disable app_list.txt
```
### Install
```
su -c sh debloater.sh install app_list.txt
```
### Uninstall
```
su -c sh debloater.sh uninstall app_list.txt
```
